# 📊 Student Performance Analytics

A responsive web dashboard for tracking and visualizing student academic performance across subjects.

Built as a mini project using **HTML, CSS, and JavaScript** — no frameworks, no backend required.

---

## ✨ Features

- **Overview Dashboard** — class-wide metrics: average, pass rate, highest & lowest scores
- **Subject Bar Chart** — average marks per subject visualized with Chart.js
- **Grade Doughnut Chart** — distribution of A/B/C/D/F grades across the class
- **Score Trend Line Chart** — performance of top 5 students across all subjects
- **Student Records Table** — searchable, filterable, and sortable student list
- **Top 3 Performers** — ranked leaderboard with gold/silver/bronze badges
- **Add Student** — form to add a new student with marks for all subjects
- **Delete Student** — remove any student from the records
- **Export CSV** — download all student data as a spreadsheet

No installation needed. Just open `index.html` in any modern browser.

```bash

cd student-performance-analytics
```
